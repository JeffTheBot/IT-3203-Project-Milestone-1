Key Concepts

HTML
The markup language that structures content so browsers can render text, images, and links.
Element, Tag, Attribute
Elements are defined by tags; attributes modify how elements behave or provide metadata.
Hyperlink (<a>)
The feature that interconnects documents, turning separate pages into the Web.
Semantics
Using meaningful elements (e.g., <article>, <section>) to express structure.
DOM
An in-memory tree representation of the document that scripts can read and modify.
Separation of Concerns
HTML for structure and meaning, CSS for presentation, JavaScript for behavior.
Living Standard
HTML is updated continuously by WHATWG and implemented by major browsers.

The Early Stages of HTML

HTML originated with Sir Tim Berners-Lee at CERN around 1989-1991 to share and interlink documents across different systems through hypertext. The earliest pages demonstrated how a browser, a server, and HTML worked together to form the World Wide Web.

Versions & Milestones

VersionYearHighlights
HTML 1.01993Basic tags for text, lists, links
HTML 2.01995Standardization; forms for input
HTML 3.21997Tables, scripts, multimedia support
HTML 4.011999Separation of structure (HTML) and style (CSS)

Mixing presentational tags like <font> and <center> with content created maintainability problems. The community shifted to a clear separation: HTML for structure and meaning, CSS for presentation, and JavaScript for behavior.
